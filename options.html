Allgemeine Einstellungen

Mit den folgenden Optionen können Sie festlegen wie mit Cookies auf Ihrem System allgemein verfahren werden soll. Um eine Entscheidung zu treffen sollten Sie sich die Vor- und Nachteile durchlesen.

Drittanbieter Cookies Speichern

Vorteile:

Durch das Aktivieren dieser Option verhindern Sie die Seite eines Dritten einen Cookie setzt. Durch dieses Cookies kann eine Dritte Seite feststellen wo Sie im Internet gewesen sind.

Nachteile:

Wenn Sie diese Option einschalten kann es auf manchen Seiten zu Problemen kommen, so dass diese Seite nicht richtig benutzt werden kann. Das liegt daran das manche Seiten ihre geschäfte über Drittanbieter regeln. Wie zum Beispiel das bezahlen mit PayPal oder auch das Reservieren von Karten für eine Veranstalltung. Sie sollten diese Option aber nur für speziefische Seiten ausschalten um Ihre Privatsphäre besser zu schützen.

Speichern von Cookies

Vorteile:

Mit dieser Option bestimmen sie wie lange ein Cookie auf Ihrem Computer gespeichert werden soll.
Immer: Die Cookies werden nie gelöscht.
Bis der Browser geschlossen wird: Wenn Sie den Browser schließen werden alle Cookies gelöscht.
Niemals: Es werden keine Cookies gespeichert auf Ihrem Computer.

Nachteile:

Ein gespeicherte Cookie kann von Ihrem Computer ausgelesen werden und so können von einer Website gespeicherte Informationen an andere Personen gelangen. Es können auch Cookies die für den Login bei einer Website benutzt werden von Ihrem Computer gestolen werden.

Login Cookies behalten

Vorteile:

Ein Login Cookie ist der Cookie der Sie Authentifiziert bei der Webseite bei der Sie sich bereits eingeloggt haben. Dadurch müssen Sie nicht immer wieder die login Daten eingeben um eine Webseite zu benutzen. Wenn Sie diese Option aktivieren werden die Cookies die dafür zuständig sind nicht gelöscht, durch aktionen die Sie mit dieser Erweiterung ausführen können. Dadurch haben Sie den Vorteil, dass Sie sich nicht immer wieder bei einer Webseite anmelden müssen und gleichzeitig keine anderen Daten durch Cookies über Sie gesammelt werden können.
Hinweis: Diese Option ist nur für manche Webseiten verfügbar.

Nachteile:

Wenn Sie diese Option aktivieren bleiben die Cookies, die Sie auf einer Webseite nach dem anmelden Authentisieren, auf Ihrem System erhalten. Dadurch könnten Dritte die Ihren Computer benutzen, sich ohne Anmeldedaten, zugang zu Ihren Daten verschaffen die auf Webseiten hinterlegt sind, die Sie benutzen.

Einstellungs Assistent

Mithilfe dieses Assistenten der Ihnen eine reihe von Fragen Stellt können sie auf einfach Art und weise Ihre Einstellungen für die Cookie Verwaltung auf Ihrem System vornehmen.

Wollen Sie Cookies auf Ihrem System speichern?

Wenn Sie es erlauben (Ja) Cookies auf Ihrem System zu speichern ermöglichen Sie es Webseiten Daten über Sie zu sammeln. Wenn Sie es nicht erlauben (Nein) Cookies auf Ihrem System zu Speichern könnte es sein das manche Webseiten Probleme haben Ihnen bestimmte Services anzubieten. Ein Cookie bietet Einer Webseite nähmlich die möglichkeit Informationen zu Speichern wie z.B. was Sie bis jetzt in Ihrem Einkaufswagen haben oder ob Sie bereits angemeldet sind. Ohne Cookies wäre es nicht möglich einen solche Funktionalität anzubieten.
