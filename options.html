Ihre Coockie Einstellungen

Hier können Sie ihre persöhnlichen Cookie Einstellungen anpassen.

  • Allgemeine Einstellungen
  • Seitenspeziefische Einstellungen

Allgemeine Einstellungen

Mit den folgenden Optionen können Sie festlegen wie mit Cookies auf Ihrem System allgemein verfahren werden soll. Um eine Entscheidung zu treffen sollten Sie sich die Vor- und Nachteile durchlesen.

Drittanbieter Cookies Speichern

Vorteile:

Durch das Aktivieren dieser Option verhindern Sie die Seite eines Dritten einen Cookie setzt. Durch dieses Cookies kann eine Dritte Seite feststellen wo Sie im Internet gewesen sind.

Nachteile:

Wenn Sie diese Option einschalten kann es auf manchen Seiten zu Problemen kommen, so dass diese Seite nicht richtig benutzt werden kann. Das liegt daran das manche Seiten ihre geschäfte über Drittanbieter regeln. Wie zum Beispiel das bezahlen mit PayPal oder auch das Reservieren von Karten für eine Veranstalltung. Sie sollten diese Option aber nur für speziefische Seiten ausschalten um Ihre Privatsphäre besser zu schützen.

Speichern von Cookies

Vorteile:

Mit dieser Option bestimmen sie wie lange ein Cookie auf Ihrem Computer gespeichert werden soll.
Immer: Die Cookies werden nie gelöscht.
Bis der Browser geschlossen wird: Wenn Sie den Browser schließen werden alle Cookies gelöscht.
Niemals: Es werden keine Cookies gespeichert auf Ihrem Computer.

Nachteile:

Ein gespeicherte Cookie kann von Ihrem Computer ausgelesen werden und so können von einer Website gespeicherte Informationen an andere Personen gelangen. Es können auch Cookies die für den Login bei einer Website benutzt werden von Ihrem Computer gestolen werden.
